O que é HTML?

HTML é a linguagem de marcação utilizada para estruturar páginas na web.

O que é CSS?

CSS é uma linguagem usada para definir o estilo e o layout das páginas web.

O que é JavaScript?

JavaScript é uma linguagem de programação usada para criar interatividade nas páginas web.

O que é uma API?

API é um conjunto de regras que permite que programas diferentes se comuniquem entre si.
